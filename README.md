Projeto Contatos
Nome do Projeto
Projeto Contatos - Aplicativo de Gerenciamento de Contatos
Objetivo do App
Este aplicativo tem como objetivo fornecer uma solução completa para gerenciamento de contatos pessoais e profissionais. Os usuários poderão adicionar, visualizar, editar e excluir contatos de forma intuitiva e eficiente, mantendo todas as informações organizadas em um único lugar.
Tecnologias Utilizadas
Angular (v20.0.0) - Framework principal para desenvolvimento da aplicação
Ionic Framework (v8.0.0) - Framework para criação de interfaces mobile híbridas
TypeScript (v5.9.0) - Linguagem de programação utilizada
Firebase (v12.6.0) - Backend as a Service para autenticação e banco de dados
Angular Fire (v20.0.1) - Biblioteca oficial para integração Angular + Firebase
Capacitor (v7.4.4) - Runtime nativo para aplicações web
RxJS (v7.8.0) - Biblioteca para programação reativa
Ionicons (v7.0.0) - Biblioteca de ícones
